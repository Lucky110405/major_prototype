# CMM BI Frontend

This is the React frontend for the Collaborative Multi-Modal Agentic BI Framework.

## Setup

1. Navigate to the frontend directory:
   ```
   cd frontend
   ```

2. Install dependencies:
   ```
   npm install
   ```

3. Start the development server:
   ```
   npm start
   ```

The app will run on http://localhost:3000 and proxy API requests to http://localhost:8000.

## Features

- **Home**: Overview of the framework
- **Ingest**: Upload files for processing
- **Query**: Search ingested data
- **Agents**: Run agentic analyses

## Technologies

- React
- React Router
- Bootstrap
- Axios
